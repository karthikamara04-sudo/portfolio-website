Accessibility in My Projects

All my projects follow these accessibility standards:

Semantic HTML5

Using proper semantic tags like <header>, <nav>, <main>, <article>, and <footer> for document structure.

ARIA Labels & Roles

Implementing ARIA attributes for screen reader support and complex interactive components.

Keyboard Navigation

All interactive elements are fully keyboard accessible with visible focus indicators.

Color Contrast

Ensuring WCAG AA (or AAA) color contrast ratios for all text and interactive elements.
